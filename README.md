# 🎯 my_learning

A public repository documenting my self-directed learning journey across two parallel tracks: **AI/ML Engineering** and **Robotics/Mechatronics**. Every folder here represents a tool, concept, or project I am actively working through — with notes, code, and exercises committed as I go.

This repo is both a portfolio and a personal knowledge base. It grows continuously, at my own pace.

---

## 🗂️ Structure

```
my_learning/
├── AI_ML/                  # Machine learning, deep learning, MLOps, LLMs
├── Robotics/               # Electronics, embedded systems, ROS 2, CAD, control systems
└── Projects/               # Applied projects that combine skills from both tracks
```

## 📌 How This Repo Works

- **Each tool folder** contains notes (`README.md`), code, notebooks, and exercises
- **Commits are daily** — progress is logged consistently, not in bursts
- **No deadlines** — depth and understanding over speed
- **Notion** is used alongside this repo to track learning status, resources, and revision cycles
